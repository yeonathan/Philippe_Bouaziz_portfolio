Projets Portfolio & Fun

InsightCRM

Application d’analyse de ventes à partir d’un fichier CSV, conçue pour les équipes commerciales. Elle permet de calculer automatiquement le chiffre d'affaires par produit et de générer des graphiques interactifs pour des rapports rapides.

Beauty_AI

Une application web légère et sans serveur pour personnaliser l’expérience beauté. Elle détecte le teint d’un visage à partir d’une photo pour suggérer une palette de maquillage cohérente. Technos: HTML / CSS / JS, extraction de caractéristiques visuelles.
